# wireframe

**WIP** - minimal wireframing css-framework. Example index.html is given.

How to develop:

* clone the repo
* run `npm install`
* run `gulp`
* read the source files
* add styles to `lib/` directory and not inside `css/` or `min/`
* add corresponding html for styles introduced
